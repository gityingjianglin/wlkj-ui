> A Vue.js 2.0 UI Toolkit for Web.

## Install
```shell
npm install wlkj-ui -S
```

## Quick Start
``` javascript
import Vue from 'vue'
import Wlkj from 'wlkj-ui'

Vue.use(Wlkj)

// or
import {
  Select,
  Button
  // ...
} from 'wlkj-ui'

Vue.component(Button.name, Button)
```

## Dev
```shell
npm run dev
```
